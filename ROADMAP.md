# ChronosDB - Roadmap for v6.0.0 and Beyond

## ✅ Completed (v5.0.0)
- [x] Core temporal graph storage
- [x] Time-travel queries (AS OF, BETWEEN)
- [x] 4 ML forecasting models
- [x] CSV/JSON/SQL import
- [x] REST + gRPC APIs
